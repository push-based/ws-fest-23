# Push-Based Angular Perf Workshop

## Installation Instructions

**System Requirements**

* `node 18`
* `npm 8`

**Clone and install**

```bash
git clone https://github.com/push-based/ws-perf-ngbe-230323.git

cd ws-perf-ngbe-230323
npm install
```

**Run**

```bash
npm run start
```
